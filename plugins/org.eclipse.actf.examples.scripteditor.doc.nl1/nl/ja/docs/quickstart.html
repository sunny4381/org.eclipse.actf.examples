概要

近年、Web上では動画コンテンツの利用が急速に拡大しています。 しかしながら、これらの動画に対して音声ガイドが提供されていることはほとんどなく、視覚障害者にとってはアクセスが出来ない、見ても情報を得ることが出来ないという問題が生じています。

動画に対する音声ガイドの付与率が非常に低い理由として、音声ガイドの台本を作成するための手間が大きく、 さらに、音声ガイドを録音するためには熟練したナレータとスタジオ等の設備が必要になるなど、作成のための敷居が高いという課題が挙げられます。

ACTF Script Editorは、入力したテキストを合成音声で読上げさせることにより、 動画を見ながら気軽に音声ガイドの編集を行うことを目的とした音声ガイド編集ツールです。 音声ガイドのスピードや合成音声の性別等のパラメータを変えながら、リアルタイムに音声ガイドを 確認・編集可能にすることで、音声ガイド作成のための手間とコストを抑えることを目指しています。

このセクションでは、Scirpt Editorの基本的な動作について紹介します。

1. Script Editorの概要

図1：Script Editor 全体図

Script Editorは、４つのパネルに分かれています（図１）。 ４つのパネルの役割は、それぞれ下記のとおりです。

  • (1)：ブラウザ： 再生したい動画のあるHTMLを表示するためのブラウザ
  • (2)：タイムラインパネル： 動画の音声、および、音声ガイドの音声のタイムラインを表示するパネル
  • (3)：音声ガイド入力パネル： 音声ガイドを入力するためのパネル
  • (4)：音声ガイドリストパネル： 既についている音声ガイドのリストを表示するパネル

インターネット上の動画ではなくPC上に保存された動画を利用する場合は、PC上に保存された動画に対する音声ガイドの付与についてを参照ください。

2. Script Editorを用いた音声ガイドのつけ方

2-1. 台本の入力

Script Editorを利用して動画を閲覧しながらガイドを作成していきます。

図２：　タイムラインパネル

タイムラインパネル内の再生・停止ボタン（図２(a)）で動画の再生・停止を行うことが出来ます。

次に、再生した音声の波形が音声タイムライン表示部（図２(b)）に表示されます。音声ガイドは基本的には無音部分、無声部分に入れますので、この波形の 大小が参考になります。 音声ガイドの入力タイミングを決めたら、音声ガイド入力パネル（図１(3)）にフォーカスを移します。

図３：　音声ガイド入力パネルの左部分

音声ガイド入力パネル（図３）では、音声ガイドをテキスト入力することが出来ます。 音声ガイド入力パネルの入力部（図３(b)）に、音声ガイドで読上げたい内容を入力します。開始の時間は同じパネルの開始・終了時間の表示部（図３(a)）で確認できます。 この時間は、タイムラインパネルのタイムスクロールバー（図２(d)）が示している時間と同期しています。入力した内容は、Previewボタン（図３(e)）を押すと、合成音声を用いて正しく読んでいるかどうかが確認できます。 この際、音声ガイドの読み上げにかかった時間が、音声ガイドのタイムライン表示部（図２(c)）に現れる四角い緑色の領域で確認できます。この四角い緑色の部分の左端が該当の音声ガイドの開始位置、右端が終了位置を表しています。 また、音声ガイドを入力すると、音声ガイド入力パネルにガイド行追加ボタン（図３(c)）と入力キャンセルのボタン（図３(d)）が表示されます。ガイド行追加ボタンを押すことで、音声ガイドの付与が実行されます。

図４：音声ガイドリストパネル

音声ガイドリストパネル（図４）には、音声ガイドの開始時間と内容が時系列順に追加されていきます。 リスト内で音声ガイドの間違いなどを見つけた場合は、リスト上で該当項目を選択することで、音声ガイド入力パネル（図３）に該当する音声ガイドを表示することが出来ます。 次に、必要に応じて、入力部（図３(b)）に表示されている音声ガイドの編集や、音声ガイド入力パネルの開始・終了時間表示部（図３の(a)）で開始時間の修正を行います。開始時間の修正は、タイムライン表示部に表示される領域をドラッグ&ドロップすることでも実施できます。 修正作業終了後、音声ガイド入力パネル上のUpdateボタンを押すと、修正された音声ガイドが再び音声ガイドリストパネル（図４）のパネルに反映されます。 選択した音声ガイドが不要である場合は、音声ガイド入力パネル上のDeleteボタンを選択してガイドを削除することも出来ます。この場合、音声ガイドリストパネル（図４）から該当項目が削除されます。

音声ガイドの読み上げに利用する合成音声については、速度や音量などを調整することも可能です。 これらの合成音声に関するパラメーターの詳細設定については、各パネルの詳細説明を参照ください。

2-2. 音声ファイル（WAVファイル）の登録 (option)

上記の手順で音声ガイドを入力後、別途作成した音声ファイル（WAVファイル）を指定して読み上げさせることも可能です。あらかじめ録音した音声で音声ガイドを付与したいときに使います。

図５：WAVファイル選択パネル

音声ガイドをつける位置が決まったらwavファイル選択のタブメニューを選んで、音声ガイド入力パネルのWAVファイル選択パネル（図５）を表示します。

WAVファイル選択パネル内のWAVファイルOPENボタンを選択すると、ファイル選択ダイアログが表示されますので、あらかじめ録音した音声ガイドのWAVファイルを選択します。 WAVファイルが登録されると、音声ガイド入力パネルのWAV File表示部にファイルの情報が表示されます。（図5） Wavファイルの内容を音声で確認したい場合は音声ガイド入力パネルのPreviewボタンを押下してください。

選択したWAVファイルで問題がなければ、音声ガイド入力パネルのUpdateボタン（図５）を選択し、音声ガイドリストに登録します。

図６：音声ガイドのリスト（WAVファイル利用時）

音声ガイドリストパネルのWAV列（図６(a)）に音符マークが表示されている場合、該当部分の音声ガイドはWAVファイルを優先して再生します。 タイムラインパネルの音声ガイドのタイムライン表示部においても、音声ガイド表示部分（緑色の部分）に音符マークが表示されます。（図７）

2-3. 音声ガイドのファイルの保存及び共有について

前述の手順で作成した音声ガイド情報の保存を行います。 Script Editorの上部に配置されたメニューから、「ファイル」->「保存」を選択すると、新規で作成していた場合は、保存のためのファイル指定のダイアログが表示され、既に作成したファイルを開いて作業をしていた場合は、上書き保存が実施されます。 同様にメニューから「ファイル」->「名前をつけて保存」、では別の名前で保存をすることが出来ます。保存されるファイルの形式はXML形式です。 この音声ガイドファイルを共有することで、他の人と音声ガイドの情報を共有することも可能です。

音声ガイドファイル共有時の注意点

音声ガイド情報を保存するXMLファイル内には、作業を行った時にHTMLが存在したフォルダ（Path）の情報が書き込まれています。 他の人と音声ガイドファイルを共有する場合は、動画ファイルおよびHTMLファイルも一緒に共有してください。
この際、音声ガイドファイル、動画ファイルおよびHTMLファイルは、現在作業しているPCと同じフォルダ構成で配置することを推奨します。

HTMLファイルが異なるフォルダ(Path)に保存されている場合には、音声ガイドファイルを下記の手順に従って書き換えてください。

  1. 音声ガイド情報をtest.xmlというファイルに保存します。この時にScriptEditorで使用していたHTMLのファイルは c:\tmp\test.html であったとします。
  2. 1.　で保存した音声ガイドファイル（test.xml）を他のPCの作業フォルダーにコピーしたとします。このとき、対象となるHTMLは c:\audiodesc\test.html に置かれていたとします。
  3. 2.　でコピーされた音声ガイドファイルをノートパッドなどのエディターで開きます。
  4. XMLファイルの4行目に <targetSite で始まる行があるので、その属性値のuriを書き換えます。
    今回の例では
    　<targetSite uri="file:///C:/tmp/test.html">
    の部分を
    　<targetSite uri="file:///C:/audiodesc/test.html">
    に　書き換えます。
  5. このように書き換えたファイルは、ScriptEditorのメニューの一番左にある「ファイル」->開くを押下して現れるダイアログボックスから選択することができます。

3. CSVを使用した音声ガイドのエクスポート/インポート

Script Editorで作成した音声ガイドはCSVファイルにも保存することが可能です。 Script Editorの上部に配置されたメニューから、「ファイル」→「CSV入力」を選択すると、CSV形式で保存するためのダイアログが出てきますので、 適切な名前をつけて保存します。 また、既にCSVで作成されているファイルを読み込んで、確認することも可能です。 同様にメニューから、「ファイル」→「CSV出力」を選択すると、CSVファイルを読み込みます。 CSVのフォーマットの詳細に関しては、CSVのフォーマットについてを参照してください。
