Installation Guide for OpenDocument Accessibility mode

Step 1

  • Install OpenOffice.org 2.x (later than 2.1.0). Launch OpenOffice.org and finish user registration.

Step 2

  • Launch aDesigner.

Step 3

  • If this installation guide does not appear when switching to OpenDocument Accessibility mode, then the OpenDocument function has been installed successfully.
