ODF アクセシビリティチェックリスト　（ODF1.0, 1.1に対応）

このチェックリストはODF 1.0 または 1.1 の文書をアクセシブルにするためのチェックリストです。このチェックリストでは以下のガイドラインを参照しています。

    ODF 1.0 Specification

  • ODF 1.0 は2005年5月1日に標準化団体OASISの標準として認定されました。 また、ISO, IECの標準(ISO/IEC26300:2006)としても認定されました。 ODF 1.0 の仕様は、OASIS ODF Technical Committeeのページから参照することができます。

    • ODF 1.1 Specification

  • ODF 1.1は2007年2月2日に標準化団体OASISの標準として認定されました。このバージョンにおいては、いくつかのアクセシビリティに関する機能が追加されました。 ODF 1.1 の仕様は、OASIS ODF Technical Committeeのページから参照することができます。

    • Accessibility Guidelines for Implementations of OpenDocument Format v1.1 (ODF Implementation Guideline)

  • この文書は、ODF 1.1に対応したアプリケーションの開発者のためのガイドラインです。OASIS ODF Accessibility Subcommitteeのページから参照することができます。
ODFのガイドライン以外に、以下のWebアクセシビリティガイドラインについても、各チェック項目が対応する記述を参照しています。

    WCAG 1.0

  • W3C Web Accessibility Initiativeによって公開されているWebアクセシビリティガイドラインです。

    • Section 508

  • アメリカ合衆国の法律で、連邦政府が調達する製品、提供する情報や、サービスが障害者にとってもアクセスできるものであることを義務づけています。

    • IBM Web Accessibility Checklist

  • IBMが公開しているWebコンテンツをアクセシブルにするためのチェックリストです。

画像に関するチェック項目

    Guideline 1.1

  • 画像に対しては、代替テキストまたはキャプションを設定してください（意味の無い画像である場合には代替テキストとして、スペース文字を設定してください）。

    • 注意

    現在の OpenOffice.org では、キャプションと画像を関連付ける機能はサポートされていません。 代替テキストを設定して、対処してください。
    また現在の OpenOffice.org Calc, Impress では、画像に対する代替テキストの機能が提供されていないため、 OpenDocument Spreadsheet (ODS), OpenDocument Presentation (ODP) ファイルに対しては、この問題は修正できません。

    関連のあるガイドライン

    • ODF 1.1 Specification, 9.2.15 Common Drawing Shape Attributes
    • ODF 1.1 Specification, 9.2.20 Title and Description
    • ODF Implementation Guideline, 3.2 Provision of alternative text versions of document content.
    • WCAG 1.0 1.1
    • Section 508 (a)
    • IBM Guideline 1

    OpenOffice.org Writer での修正方法 (代替テキストの追加)

    OpenOffice.org Writer では、画像をダブルクリックすることにより画像のプロパティダイアログが表示されます。 画像のプロパティダイアログの”オプション”タブを選択し、”次候補の名前”に代替テキストを設定してください。

    注意

    現在の OpenOffice.org では、空白の代替テキストはサポートされていません。意味の無い画像である場合には、スペース文字を設定してください。

    IBM Lotus® SymphonyTM での修正方法 (代替テキストの追加)

    IBM Lotus® SymphonyTM では、画像を選択し、右クリックメニューの”Graphic Properties”を選択することで、画像のプロパティダイアログが表示されます。 画像のプロパティダイアログの”Alternative Text”タブを選択し、”Alternative Text”に代替テキストを設定してください。

    注意

    現在の IBM Lotus® SymphonyTM では、空白の代替テキストはサポートされていません。意味の無い画像である場合には、スペース文字を設定してください。

    IBM Lotus® SymphonyTM での修正方法 (キャプションの追加)

    IBM Lotus® SymphonyTM では、画像を選択し、右クリックメニューの”Add Caption”を選択することで、キャプションを設定することができます。

    Guideline 1.2 (ODF 1.1以降が対象)

  • 複雑な画像を用いている場合には、詳細な説明文を設定することを検討してください。

    • 関連のあるガイドライン

    • ODF 1.1 Specification, 9.2.20 Title and Description
    • ODF Implementation Guideline, 3.2 Provision of alternative text versions of document content.
    • WCAG 1.0 1.1
    • Section 508 (a)
    • IBM Guideline 3

    IBM Lotus® SymphonyTM での修正方法

    IBM Lotus® SymphonyTM では、画像を選択し、右クリックメニューの”Graphic Properties”を選択することで、画像のプロパティダイアログが表示されます。 画像のプロパティダイアログの”Alternative Text”タブを選択し、”Long Description”に説明文を設定してください。

    注意

    現在の IBM Lotus® SymphonyTM では、空白の説明文はサポートされていません。説明文が必要ない場合には、スペース文字を設定してください。

テーブルに関するチェック項目

    Guideline 2.1

  • テーブルに対しては、テーブルヘッダーを設定してください。

    • 関連のあるガイドライン

    • ODF Implementation Guideline, 3.1.6 Table navigation.
    • WCAG 1.0 5.1
    • Section 508 (g)
    • IBM Guideline 10

    OpenOffice.org での修正方法

    OpenOffice.org では、テーブルの中で右クリック後、メニューの”表”を選択することで、表の書式ダイアログが表示されます。 表の書式ダイアログの”体裁”タブで、"ヘディングを繰り返す"のチェックボックスをオンにすることで、表のヘッダーを設定することができます。

    IBM Lotus® SymphonyTM での修正方法

    IBM Lotus® SymphonyTM では、テーブルの中で右クリック後、メニューの”Table Properties”を選択することで、表の書式ダイアログが表示されます。 Table Properties ダイアログの”Rows and Columns”タブで、"Column header" または "Row label" のチェックボックスをオンにすることで、表のヘッダーを設定することができます。

    Guideline 2.2

  • テーブルに対しては、キャプションを設定することを検討してください。

    • 注意

    現在のODFでは、テーブルとキャプションを関連づける機能はサポートされていません。テーブルとキャプションの間に文章を挿入しないようにしてください。

    関連のあるガイドライン

    • Section 508 (g)
    • IBM Guideline 10

    OpenOffice.org での修正方法

    OpenOffice.org では、テーブルの中で右クリック後、メニューの”図表番号”を選択することで、キャプションを設定することができます。

    IBM Lotus® SymphonyTM での修正方法

    IBM Lotus® SymphonyTM では、テーブルの中で右クリック後、メニューの”Add Caption”を選択することで、キャプションを設定することができます。

イメージマップに関するチェック項目

    Guideline 3.1

  • イメージマップに対しては、代替テキストを設定してください。

    Guideline 3.2 (ODF 1.1以降が対象)

  • イメージマップに複雑な画像を用いている場合には、詳細な説明文を設定することを検討してください。

    • 関連のあるガイドライン

    • ODF 1.1 Specification, 9.3.10 Client Side Image Maps
    • ODF Implementation Guideline, 3.2.1 Image maps.
    • WCAG 1.0 1.1
    • Section 508 (a)
    • IBM Guideline 2

リンクに関するチェック項目

    Guideline 5.1 (ODF 1.1以降が対象)

  • リンクに対しては、ヒントテキストによって、リンク先に関する説明文を設定してください。

    • 関連のあるガイドライン

    • ODF 1.1 Specification, 5.1.4 Hyperlinks
    • WCAG 1.0 13.1

    IBM Lotus® SymphonyTM での修正方法

    IBM Lotus® SymphonyTM では、リンクテキストを選択し、メニューの"Create" > "Hyperlink"を選択してください。 Hyperlink ダイアログの"Description"にリンク先に関する説明文を設定してください。

フォームに関するチェック項目

    Guideline 7.1

  • フォームコントロールに対しては、ラベルを設定してください。

    • 関連のあるガイドライン

    • ODF 1.1 Specification, 11.5.9 Label
    • WCAG 1.0 12.4
    • Section 508 (n)
    • IBM Guideline 7

    OpenOffice.org での修正方法

    OpenOffice.org では、フォームコントロールを選択して右クリック後、メニューの”コントロール”を選択することで、属性ダイアログを表示することができます。 属性ダイアログの”全般”タブを選択し、”ラベルフィールド”項目のボタンをクリックすることで、ラベルフィールドの選択ダイアログを表示することができます。 Label Field Selectionダイアログで、ラベルフィールドを選択することにより、フォームコントロールにラベルを設定することができます。

    IBM Lotus® SymphonyTM での修正方法

    IBM Lotus® SymphonyTM では、フォームコントロールを選択して右クリック後、メニューの”Control”を選択することで、属性ダイアログを表示することができます。 属性ダイアログの”General”タブを選択し、”Label Field”項目のボタンをクリックすることで、ラベルフィールドの選択ダイアログを表示することができます。 ラベルフィールドの選択ダイアログで、ラベルフィールドを選択することにより、フォームコントロールにラベルを設定することができます。

ナビゲーションオーダーに関するチェック項目

    Guideline 8.1 (ODF 1.1以降が対象)

  • プレゼンテーションスライド中のオブジェクトに対して、ナビゲーションオーダーが正しく設定されているか確認してください。

    • 関連のあるガイドライン

    • ODF 1.1 Specification, 9.1.4 Drawing Pages
    • WCAG 1.0 9.4

    IBM Lotus® SymphonyTM での修正方法

    IBM Lotus® SymphonyTM では、メニューの "Presentation" > "Object Navigation Order" を選択することで、ナビゲーションオーダーを設定するダイアログを表示することができます。 "Object Navigation Order"ダイアログで、オブジェクトの順番を 並び替えることによって、ナビゲーションオーダーを設定することができます。

    Checkpoint 8.2

  • フォームコントロールに対して、ナビゲーションオーダーが正しく設定されているか確認してください

    • 関連のあるガイドライン

    • ODF 1.1 Specification, 11.5.15 Tab Index
    • WCAG 1.0 9.4
    • IBM Guideline 7

    OpenOffice.orgでの修正方法

    OpenOffice.org では、フォームコントロールを選択して右クリック後、メニューの”コントロール”を選択することで、属性ダイアログを表示することができます。 属性ダイアログの”全般”タブを選択し、”タブの順序”項目のボタンをクリックすることで、タブの順序ダイアログを表示することができます。 タブの順序ダイアログで、コントロールの順番を並び替えることによって、ナビゲーションオーダーを設定することができます。

    IBM Lotus® SymphonyTM での修正方法

    IBM Lotus® SymphonyTM では、フォームコントロールを選択して右クリック後、メニューの”Control”を選択することで、属性ダイアログを表示することができます。 属性ダイアログの”General”タブを選択し、”Order”項目のリストボックスの数値を設定することでナビゲーションオーダーを設定することができます。

    Checkpoint 8.3

  • フォームコントロールに対して、タブストップ機能を有効にすることを検討してください

    • 関連のあるガイドライン

    • ODF 1.1 Specification, 11.5.16 Tab Stop
    • WCAG 1.0 9.4
    • IBM Guideline 7

    OpenOffice.org での修正方法

    OpenOffice.org では、フォームコントロールを選択して右クリック後、メニューの”コントロール”を選択することで、属性ダイアログを表示することができます。 属性ダイアログの”全般”タブを選択し、”タブストップ”リストボックスの値を”はい”に設定することで、タブストップ機能を有効にすることができます。

    IBM Lotus® SymphonyTM での修正方法

    IBM Lotus® SymphonyTM では、フォームコントロールを選択して右クリック後、メニューの”Control”を選択することで、属性ダイアログを表示することができます。 属性ダイアログの”General”タブを選択し、”Tabstop”リストボックスの値を”Yes”に設定することで、タブストップ機能を有効にすることができます。

    Guideline 9.1

  • ODF 1.1 の機能が使われていますが、ODF のバージョンが 1.0 として保存されています。aDesignerでは、ODF 1.1 のファイルとしてアクセシビリティチェックを行います。

    注意

    OpenOffice.org 2.3.0 と IBM Lotus® SymphonyTM Beta 1 では、ODF 1.1 の機能を使用していますが、ODFのバージョンが 1.0 として保存されます。
