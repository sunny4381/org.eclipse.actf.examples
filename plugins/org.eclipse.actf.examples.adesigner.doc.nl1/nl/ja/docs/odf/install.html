「OpenDocument アクセシビリティモード」 のインストールガイド

ステップ1

  • OpenOffice.org をインストールしてください (バージョン2.x (2.1.0以上)をサポートしています)。　OpenOffice.org を起動してユーザー登録を行ってください。

ステップ2

  • aDesigner を起動してください。

ステップ3

  • OpenDocument アクセシビリティモードに切り替えて、このインストールガイドが自動的に表示されなければ、インストール完了です。
