Flash アクセシビリティモードの概要

近年のウェブサイトにおいては、Ajax (DHTML) や Adobe® Flash (以下 Flash) などの技術で、ビジュアルに優れ、ユーザの操作によって動的に変化する、リッチなウェブコンテンツを簡単に作成することが可能になりました。その反面、これらのコンテンツが、視覚障害者にとって極めてアクセスしにくいという状況を引き起こしています。 しかし、アクセシブルな Flash コンテンツの制作を支援するシステムは、これまで存在していませんでした。

aDesigner の 「Flash アクセシビリティモード」 は、Flash コンテンツを内部に含むウェブページのアクセシビリティ・チェックを支援するためのモードです。このモードでは GUI アクセシビリティモードと同様、Microsoft® Windows® の支援技術 (assistive technology) である Microsoft Active Accessibility (MSAA) による出力を利用することにより、ウェブページがスクリーンリーダによってどのように読み上げられるかを、 実際にリーダを動作させることなく確認することができます。また、aDesigner は Flash コンテンツの実行時の内部構造を取得することができます。これは、Flash コンテンツの制作者が、アクセシビリティ・チェックの結果を元に修正を行う場合に有用です。
