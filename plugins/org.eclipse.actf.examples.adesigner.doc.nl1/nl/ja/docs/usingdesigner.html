aDesignerの使い方

  • aDesigner のモードを変更する
  • 「HTML アクセシビリティモード」
    • 音声インターフェースユーザーのアクセシビリティやユーザビリティをチェック
    • ロービジョンユーザのアクセシビリティやユーザビリティをチェック
    • 「アクセシビリティ検証・音声ユーザビリティ視覚化」 で検出されるアクセシビリティ、およびユーザビリティの問題点
    • 「ロービジョンシミュレーション」で検出されるアクセシビリティ、およびユーザビリティの問題点
    • Web アクセシビリティ・ガイドラインへの準拠
  • 「OpenDocument アクセシビリティモード」
    • OpenDocument アクセシビリティモードのインストールガイド
    • 音声インターフェースユーザーのアクセシビリティやユーザビリティをチェック
    • ロービジョンユーザーのアクセシビリティやユーザビリティをチェック
    • 会議室でのスライドショーの見え方をチェック
    • OpenDocument のアクセシビリティチェックリスト
  • 「GUI アクセシビリティモード」 でアプリケーションのアクセシビリティをチェック
  • 「Flash アクセシビリティモード」 で Flash コンテンツを含むウェブページのアクセシビリティをチェック
