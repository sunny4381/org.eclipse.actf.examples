HTML 文書表示パネル

HTML 文書表示パネルは、現在処理中のウェブページの HTML 文書のソースファイルを表示するものです。 以下の手順で開くことができます。

  • 詳細レポートの個別表示の中から1行を選択し、右クリックのコンテキストメニューから 「ソース HTML を見る」 を選択する。
HTML 文書表示パネルでは、詳細レポートで選択した項目に対応する HTML 要素がハイライトされます。


HTML 文書表示パネルでは、ハイライトされている HTML 要素のカーソル位置の行番号、およびその行の先頭から数えた文字位置も表示されます。 これにより、HTML 文書の中の修正が必要な場所がすぐに分かります。

「検索」 ボタンを選択すると、HTML 文書内を検索するためのパネルが開きます。
