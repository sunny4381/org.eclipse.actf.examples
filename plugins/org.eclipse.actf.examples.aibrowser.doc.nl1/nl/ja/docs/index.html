Accessibility Internet Browser for Multimedia ヘルプ

Accessibility Internet Browser for Multimedia (aiBrowser for Multimedia) は、動画等のマルチメディアをコントロールする機能と、ユーザが注釈を追加して作ることのできる代替インタフェースを提供する、 視覚障害者のためのインターネットブラウザです。
