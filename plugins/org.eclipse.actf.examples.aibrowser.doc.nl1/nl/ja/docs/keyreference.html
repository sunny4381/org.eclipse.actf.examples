すべてのショートカットキー

ショートカットキーに割り当てられている機能はメニューでも操作できます。

  • ブラウザ・コントロール 1（ファイル・メニュー）
  • ブラウザ・コントロール 2（ウィンドウ・メニュー）
  • ナビゲーション機能（ナビゲーション・メニュー）
  • ジャンプ機能（ジャンプ・メニュー）
  • マルチメディア・コントロール（マルチメディア・メニュー）
  • 注釈（注釈メニュー）

ブラウザ・コントロール 1 （ファイル・メニュー）

Ctrl + O
「URL を開く」 ダイアログを開く。開きたいページの URL を入力してください
Ctrl + Alt + I
現在のページを通常使用するブラウザで開く
Ctrl + W
現在のページを閉じる
Ctrl + I
新しいメタデータ・ファイルを読み込む。 メタデータ・ファイルを読み込み、aiBrowserで利用可能にします。
Ctrl + E
現在のページの注釈情報をメタデータ・ファイルとして書き出す。 現在のページに注釈情報がない場合は、書き出しは行われません。
Ctrl + Alt + E
保存されている全ての注釈情報を書き出す。 　aiBrowserに保存されている全ての注釈情報を圧縮されたファイル形式で書き出します。

ブラウザ・コントロール 2 （ウィンドウ・メニュー）

F5
ページ更新。現在のページを最新の情報に更新します
Alt + 右矢印キー
次に進む。次のページ履歴へ移動します
Alt + 左矢印キー
前に戻る。前のページ履歴へ移動します
Alt + D
アドレス入力バーへ移動
Ctrl + Tab
次のタブへ移動
Ctrl + Shift + Tab
前のタブへ移動
S
現在のモードを読み上げ
T
現在のメディア状態を読み上げ
Ctrl + Alt + T
ナビゲーション・ツリービューと音声解説の表示・非表示を切替

ナビゲーション機能 (ナビゲーション・メニュー)

下矢印キー
次の項目へ移動
上矢印キー
前の項目へ移動
Alt + 下矢印キー
現在位置から最後までを読み上げ
Enter または スペース
選択項目をクリック / チェックボックスやラジオボタンの選択を切替 / テキスト入力ダイアログを開く
Ctrl + F
「文字列の検索」 ダイアログを開く。Enter キーを押すと下方向に検索を行います
Ctrl + R
「文字列の検索」 ダイアログを開く。Enter キーを押すと上方向に検索を行います
Ctrl + Alt + K
アクセスキーのリストを開く
F4
コンテンツの再解析
Ctrl + Alt + N
メタデータの切替
Ctrl + Alt + A
音声解説機能のオン・オフの切替

ジャンプ機能（ジャンプ・メニュー）

Home または Ctrl + Home
ページの先頭へ移動
End または Ctrl + End
ページの最後へ移動
H または 右矢印キー
次の見出しへ移動
Tab
次のリンクへ移動
O
次のオブジェクト（Flash、Windows Mediaなど）へ移動
F
次のフォームへ移動
I
次のリスト項目へ移動
M
次のメディア・オブジェクトへ移動
N
次のテキストブロックへ移動
A
次の代替テキスト編集点へ移動
Shift + H または 左矢印キー
前の見出しへ移動
Shift + Tab
前のリンクへ移動
Shift + O
前のオブジェクトへ移動
Shift + F
前のフォームへ移動
Shift + I
前のリスト項目へ移動
Shift + M
前のメディア・オブジェクトへ移動
Shift + N
前のテキストブロックへ移動
Shift + A
次の代替テキスト編集点へ移動
1, 2, 3, 4, 5, 6
次のレベル（1から6）見出しへ移動
Shift + 1, 2, 3, 4, 5, 6
前のレベル（1から6）見出しへ移動

マルチメディア・コントロール (マルチメディア・メニュー)

Ctrl + P
ビデオ・オーディオの再生または一時停止
Ctrl + S
ビデオ・オーディオの停止
Pause
ビデオ・オーディオの一時停止
Ctrl + M
音を消す
Ctrl + J
音量を下げる
Ctrl + K
音量を上げる
Ctrl + Shift + J
音量を細かく下げる
Ctrl + Shift + K
音量を細かく上げる
Ctrl + Alt + Page Up
音声読み上げ速度を上げる
Ctrl + Alt + Page Down
音声読み上げ速度を下げる
Ctrl + R
Flashコンテンツの自動修正を試す。 これはFlashコンテンツを解析することにより、よりアクセシブルにすることができる機能です。 もしFlashコンテンツがアクセシブルでない場合はこの機能をお試し下さい。

注釈（注釈メニュー）

Ctrl + A
代替テキスト編集点で代替テキストを編集。代替テキスト編集点でのみ有効です
Ctrl + H
しおりの追加または削除
Ctrl + Alt + S
注釈の保管。「自動的に注釈を保管する」が設定されている場合には注釈は自動的に保管されます
Ctrl + Alt + R
現在のページ内の全注釈を削除。元に戻せません
