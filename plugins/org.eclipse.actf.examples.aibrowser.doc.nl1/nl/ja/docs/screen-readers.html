スクリーンリーダーと一緒に利用する

aiBrowser は、起動時にスクリーンリーダーを検出します。 aiBrowser を起動する前に普段使っているスクリーンリーダーを実行してください。 aiBrowserは JAWS for Windows および Window-Eyes に対応しています。 これ以外のスクリーンリーダーについては、現在対応を検討中です。

  • JAWS for Windows を利用する
  • Window-Eyes を利用する

JAWS for Windows を利用する

設定

音声設定
JAWS が実行中であれば、aiBrowser は起動時に自動的に JAWS 経由で読み上げを行うように音声設定が切り替えられます。 正しく動作していない場合は、音声設定 を手動で行ってください。

JAWS 利用時の特有の動作

画面構成
JAWS 利用時の aiBrowser 画面は、JAWS を利用しない場合と比較して大きく見た目が異なります。 ナビゲーション・ツリービュー、音声解説、およびアドレス・バーは、JAWS 実行時には表示されません。これは aiBrowser と JAWS の読み上げが衝突することを回避するためです。
すべて読み上げ（Alt + 下矢印キー）
「すべて読み上げ」実行時には、ブラウザ画面の上にテキスト読み上げ画面が表示されます。

Window-Eyes を利用する

設定

音声設定
Window-Eyes が実行中であれば、aiBrowser は起動時に自動的に Window-Eyes 経由で読み上げを行うように音声設定が切り替えられます。 正しく動作していない場合は、音声設定 を手動で行ってください。

Window-Eyes 利用時の特有の動作

すべて読み上げ（Alt + 下矢印キー）
「すべて読み上げ」はWindow-Eyesをご利用の場合には機能しません。
